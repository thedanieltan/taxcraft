# Product contract

TaxCraft calculates personal income tax for explicitly supported jurisdictions and tax years.

## Output

A successful calculation returns an estimate, a calculation breakdown, the model version, assumptions, coverage limits and official source references.

## Boundaries

TaxCraft does not retain user calculation data, accept tax documents, file returns or recommend tax actions. Cases that require unsupported legal judgment return an unsupported result rather than a guess.

## Country packages

The official service loads only country packages maintained in this repository. Other developers may implement the public package contract in separate repositories. TaxCraft does not review, list or endorse those packages.
