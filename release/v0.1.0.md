# TaxCraft v0.1.0

TaxCraft v0.1.0 is the first public release of the deterministic personal income tax engine, stateless API and reference calculator.

## Included

- public calculation request and result contracts;
- deterministic country-package runtime and SDK;
- Singapore resident personal income tax for YA 2024, YA 2025 and YA 2026;
- official-source-linked tax brackets and general rebates;
- a Singapore chargeable-income worksheet using user-confirmed totals;
- a stateless HTTP API and responsive reference calculator;
- Docker and Cloudflare Worker deployment paths;
- autonomous IRAS rate monitoring with independent extraction and fail-closed admission;
- a rolling three-tax-year lifecycle.

## Boundaries

TaxCraft does not store user calculation data and does not provide tax, legal, accounting or financial advice.

The Singapore worksheet performs arithmetic only. It does not determine residency, whether income is taxable, or whether a deduction or relief is legally available.

## Deployment status

The repository and container are implementation-accepted through CI. A public hosted deployment is a separate acceptance state and requires the configured live smoke checks to pass.
