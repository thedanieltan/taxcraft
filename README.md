# TaxCraft

TaxCraft is an open-source personal income tax calculation engine, API and reference calculator.

It applies versioned country rules to structured tax facts and returns a deterministic estimate, calculation breakdown, coverage limits and official sources.

TaxCraft does not store user calculation data and does not provide tax, legal, accounting or financial advice.

Development is organised through work-package pull requests. The initial implementation starts with Singapore and supports a rolling maximum of three tax years per jurisdiction.
